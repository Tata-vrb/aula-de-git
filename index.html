Olá mundo do Git e GitHub!


git init:Inicia um novo reppositório Git local na pasta atual

git add:Adiciona arquivos selecionados para a área de stage (pronto para ser commitado)

git commit -m:Grava as alterações feitas na área de stage como uma nova versão do código

git status:Verifica o status atual do repositório, incluindo arquivos modificados e arquivos na área de stage

git log:Ver todo o histórico de commit, com suas descrições e ids

git chekout:usado para alternar entre branches, criar novos branches, ou mudar para uma versão anterior do código em uma branch existente

git branch: Ver todas as branchs criadas do projetos

git merge:permite unir o conteúdo de duas ou mais branchs em uma única branch. É usado para integrar alterações feitas em uma branch secundária (geralmente uma branch de desenvolvimento) na brnch princupal(geralmente a branch master)

git remote add origin: usado para conectar o projeto local com o repositório do Github. Você aponta a URL do projeto no Github.

git push: comando para enviar para Github.

git clone: pegar o repositório do Github e baixar na sua máquina. Você aponta a URL do projeto no Github que quer baixar.
